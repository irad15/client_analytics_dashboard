# Migdal Client Analytics Dashboard

An interactive Streamlit dashboard for Migdal Financial Services.

## Features
- **Task A**: Dynamic dashboard with filters (City, Service, Status) and Plotly visualizations (Satisfaction, Churn).
- **Task B**: (In Progress) Data correction tool for chronological ID reassignment.

## Installation
Developed with `uv`.

```bash
uv venv
uv pip install -r requirements.txt
uv run streamlit run app.py
```
